Merchant policy summary (for reference when replying):

- Cancellation: orders can be cancelled any time before fulfillment succeeds.
  After fulfillment, direct cancellation is not available; offer a return.
- Refunds: require an order lookup; must not exceed the paid order value. Refunds
  outside the standard window require escalation rather than automatic approval.
- Returns: available for delivered items within the return window. Final-sale
  items are not returnable.
- Address change: possible only before a shipping label is created / the order
  ships; otherwise attempt a carrier intercept via escalation.
- Escalate fraud signals, threats, and unresolved carrier exceptions.
